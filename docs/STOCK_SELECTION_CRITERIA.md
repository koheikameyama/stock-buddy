# 株式銘柄選定基準

## 概要

Stock Buddyで取り扱う株式銘柄の選定基準を定義します。
この基準は、初心者向け投資アドバイスサービスとして、安全性・理解しやすさ・実用性を重視しています。

## 基本方針

**対象ユーザー**: 投資初心者（3万円〜の少額投資を検討している人）
**サービスの性質**: AIによる銘柄推奨を含むため、リスクの高い銘柄は避ける

## 選定基準

### 1. 価格帯の補完

**目的**: 様々な予算帯のユーザーに対応

- **3万円以下**: 100株で30,000円以下（株価300円以下）
- **5万円以下**: 100株で50,000円以下（株価500円以下）
- **10万円以下**: 100株で100,000円以下（株価1,000円以下）

**理由**:
- オンボーディングで「3万円」「5万円」の予算を入力した場合に推奨できる銘柄が必要
- 現在の日経225のみでは、低価格帯の選択肢が不足（3万円以下: 1-2銘柄のみ）

### 2. 市場の流動性

**基準**:
- 東証プライム市場に上場
- 1日の平均出来高: 10万株以上

**理由**:
- 実際に買いやすく、売りやすい
- スプレッド（買値と売値の差）が小さい
- 初心者が「買えない」「売れない」リスクを回避

### 3. 企業の安定性

**基準**:
- 時価総額: 1,000億円以上
- 上場年数: 10年以上
- 財務状況: 過去3年で2年以上黒字、または業界大手

**理由**:
- 初心者向けサービスのため、倒産リスクの高い銘柄は除外
- AIが推奨する以上、一定の安全性を確保
- 経営が不安定な企業は避ける

### 4. セクター分散

**基準**: 以下のセクターをバランスよくカバー

#### 必須セクター
- 金融（銀行・証券・保険）
- 通信・情報
- 小売・消費
- 製造業・機械
- エネルギー・素材
- 不動産・建設
- 運輸・物流
- 医薬品・ヘルスケア

#### 補完セクター（低価格帯に多い）
- 地方銀行: 地域経済の代表、安定配当
- 電力・ガス: インフラ、配当重視の長期投資向け
- 鉄道: 安定収益、景気敏感だが理解しやすい

**理由**:
- ポートフォリオ提案時に、1つのセクターに偏らないようにする
- リスク分散の観点から重要

### 5. 知名度・理解しやすさ

**基準**:
- 一般消費者が知っている企業・ブランド
- 事業内容が明確でわかりやすい

**良い例**:
- トヨタ自動車（自動車製造）
- セブン&アイ（コンビニ運営）
- 任天堂（ゲーム機・ソフト）
- 東京ガス（ガス供給）

**避けるべき例**:
- 持株会社で事業内容が複雑
- B2Bのみで一般には馴染みがない

**理由**:
- 初心者が「この会社は何をしているか」理解できることが重要
- 投資判断の際に、ニュースや業績を自分で理解できる

## 銘柄構成

### コア銘柄（日経225ベース）

約160銘柄を選定済み。大型株中心で、以下の特徴：

- 時価総額が大きい
- 流動性が高い
- 知名度が高い
- **課題**: 株価が高く、少額投資に向かない

### 補完銘柄（低価格帯）

約40-50銘柄を追加予定。以下の特徴：

- 100株で3-10万円で購入可能
- 東証プライム市場
- 地方銀行、電力、通信、小売など
- 安定配当が期待できる
- 初心者にも理解しやすい業種

### 合計目標

**200-250銘柄**で運用開始

- 3万円以下で買える銘柄: 15-25銘柄
- 5万円以下で買える銘柄: 40-60銘柄
- 10万円以下で買える銘柄: 80-120銘柄

## 除外基準

以下の銘柄は、基準を満たしていても除外します：

### 1. ボラティリティが極端に高い
- 過去1年で株価が50%以上変動した銘柄（ただし業界全体の影響は除く）

### 2. 流動性が極端に低い
- 1日の平均出来高が5万株未満

### 3. 財務状況が悪化している
- 3期連続赤字
- 債務超過
- 直近で大規模なリストラを実施

### 4. 社会的に問題がある
- 不祥事が発覚して間もない
- コンプライアンス違反

### 5. 複雑すぎる
- 持株会社で事業内容が多岐にわたり説明が困難
- 海外事業が主体で日本での知名度が低い

## メンテナンス方針

### 定期見直し（年2回）

- 6月: 3月決算企業の決算発表後
- 12月: 9月決算企業の決算発表後

### 見直し項目

1. 財務状況の確認（赤字転落、債務超過など）
2. 流動性の確認（出来高の推移）
3. 上場廃止・統合の確認
4. セクター構成のバランス確認

### 追加・削除の判断

**追加**:
- 新たに東証プライムに上場した優良企業
- ユーザーからの要望が多い銘柄（基準を満たす場合）

**削除**:
- 上場廃止が決定
- 財務状況が著しく悪化
- 不祥事などで社会的信用を失った

## 補足: 価格帯別の特徴

### 3万円以下（株価300円以下）

**典型的な業種**:
- 地方銀行
- 電力会社
- 一部の小売・食品

**特徴**:
- 配当利回りが高め（3-5%）
- 成長性は低いが安定
- 長期保有向き

### 5-10万円（株価500-1000円）

**典型的な業種**:
- 大手銀行
- 通信キャリア
- 小売・食品大手
- 鉄道

**特徴**:
- バランス型（配当+値上がり益）
- 初心者に人気のセクター

### 10万円以上（株価1000円以上）

**典型的な業種**:
- 製造業大手（自動車、電機）
- IT・ハイテク
- 製薬

**特徴**:
- 成長性が期待できる
- グローバル企業が多い
- 株価変動が大きめ
