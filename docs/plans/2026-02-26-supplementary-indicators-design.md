# 購入判断に補助指標を追加（KOH-197）

## 概要

購入判断AIの分析精度を向上させるため、「今このタイミングで買うべきか」の判断材料となる補助指標を3つ追加する。

- ギャップアップ率
- 出来高急増率（Volume Spike）
- 売買代金

全て既存のyfinanceデータ（OHLCV）から計算可能。新規API不要。

## 活用箇所

1. **購入判断AI** — プロンプトに補助指標としてコンテキスト追加 + 判断基準の指示
2. **銘柄詳細画面** — 初心者向け解説付きで表示
3. **強制補正ルール** — 異常値検知による安全ルール追加

## 1. DBスキーマ変更

Stockテーブルに4カラム追加：

```prisma
model Stock {
  // 新規追加
  latestOpen      Decimal?  @db.Decimal(12, 2)  // 始値
  gapUpRate       Decimal?  @db.Decimal(8, 2)   // ギャップアップ率 (%)
  volumeSpikeRate Decimal?  @db.Decimal(8, 2)   // 出来高急増率（当日/20日平均）
  turnoverValue   Decimal?  @db.Decimal(16, 0)  // 売買代金（出来高×終値、円）
}
```

### 計算式

| 指標 | 計算式 | 説明 |
|------|--------|------|
| gapUpRate | `(当日始値 - 前日終値) / 前日終値 × 100` | 前日引け→当日寄付きの値動き |
| volumeSpikeRate | `当日出来高 / 20日平均出来高` | 通常と比べた売買の活発さ |
| turnoverValue | `出来高 × 終値` | 1日の取引金額（円） |

### 既存カラムとの違い

- `volumeRatio`（既存）: 直近3日平均 / 4-30日前平均 → 短期トレンド
- `volumeSpikeRate`（新規）: 当日 / 20日平均 → 当日の異常検知

## 2. バッチ処理

`fetch_stock_prices.py` の既存処理に計算ロジックを追加。

- `hist = ticker.history(period="1mo")` で30日分OHLCV取得済み → 追加APIコールなし
- 既存の `volatility`, `volumeRatio`, `maDeviationRate` と同じ箇所で計算・保存

### エッジケース

| ケース | 対応 |
|--------|------|
| ヒストリカルデータが2日未満 | `gapUpRate = None` |
| 20日分のデータがない | 取得できた分の平均で計算 |
| 出来高ゼロ | `volumeSpikeRate = None`, `turnoverValue = 0` |

## 3. 購入判断AIプロンプト

### コンテキスト追加

既存のテクニカル分析セクションに「タイミング補助指標」を追加：

```
【タイミング補助指標】
・ギャップアップ率: +2.3%（前日引け→当日寄付きで2.3%上昇）
  → 大きなギャップアップは過熱感の兆候。ギャップダウンは売り圧力
・出来高急増率: 3.2倍（20日平均比）
  → 1.5倍以上は注目度上昇。急増は材料出現の可能性
・売買代金: 15.2億円
  → 流動性の目安。大きいほど売買しやすい
```

### 判断基準の指示追加

```
- ギャップアップ率が高い（>5%）場合、飛びつき買いのリスクを警告
- 出来高急増率が高い（>2.0倍）場合、材料の有無を確認し判断に反映
- 売買代金が小さい銘柄はスリッページリスクを考慮
```

## 4. 強制補正ルール

既存の安全ルール（`purchase-recommendation-core.ts`）に追加：

| 条件 | 修正内容 | 理由 |
|------|--------|------|
| `gapUpRate >= 10%` | buy → stay | 急騰後の飛びつき買い防止 |
| `volumeSpikeRate >= 5.0` かつ `gapUpRate >= 5%` | buy → stay | 異常な出来高+急騰 = 仕手株リスク |

### 既存ルールとの関係

- 既存: `weekChangeRate >= 30%` → 週間レベルの急騰検知
- 新規: `gapUpRate >= 10%` → 日次レベルの急騰検知（補完関係）

## 5. 銘柄詳細画面

テクニカル指標セクションに「タイミング指標」として追加。

### 表示内容

| 指標 | 表示例 | 解説 |
|------|--------|------|
| ギャップアップ率 | +2.3% | 前日引けから当日寄付きの値動き。大きいほど注目度が高いが、飛びつき買いに注意 |
| 出来高急増率 | 3.2倍 | 20日平均と比べた売買の活発さ。1.5倍超は材料が出た可能性あり |
| 売買代金 | 15.2億円 | 1日の取引金額。大きいほど売りたい時にすぐ売れる |

### 表示ルール

- 値がnullの場合は「—」を表示
- ギャップアップ率: 正は赤（上昇）、負は青（下落）
- 出来高急増率: 2.0倍以上はハイライト表示
- 売買代金: 億円単位にフォーマット（1億未満は万円単位）
- 文言はi18nで管理（`messages/ja.json`）
