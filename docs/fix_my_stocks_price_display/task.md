# 銘柄移行スクリプトのレート制限耐性強化

`migrate-existing-tickers.ts` が大量の銘柄を処理する際に Yahoo Finance のレート制限に抵触して失敗するのを防ぎ、かつ東証（.T）のみを対象とするように修正しました。

## 修正

- [x] `lib/constants.ts` からスリープ時間を参照するように修正（スクリプト内で10秒に設定）
- [x] バッチ処理間にウェイト（スリープ）を追加
- [x] レート制限エラー発生時のリトライロジック（指数バックオフ付き）を実装
- [x] クエリ条件を整理（ドットなしの銘柄を抽出し、東証 .T への検証・移行を優先）

## 検証

- [x] Pythonスクリプトの型警告修正により、副次的なエラーを防止
